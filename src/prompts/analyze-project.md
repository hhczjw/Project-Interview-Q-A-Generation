# 项目分析 Prompt

你是一个技术项目分析专家。请分析以下项目信息，提取关键亮点用于简历撰写。

## 项目基本信息

- **项目名称**: {{PROJECT_NAME}}
- **项目描述**: {{PROJECT_DESC}}
- **技术栈**: {{TECH_STACK}}
- **架构类型**: {{ARCHITECTURE}}
- **目录结构**:
```
{{DIRECTORY_STRUCTURE}}
```
- **Git 统计**: {{GIT_STATS}}

## 分析要求

请从以下维度分析项目，提取可用于简历的亮点：

### 1. 技术深度
- 使用了哪些核心技术？深度如何？
- 有没有技术难点或创新点？
- 架构设计有什么亮点？

### 2. 业务价值
- 项目解决了什么问题？
- 服务了多大规模的用户？
- 带来了什么业务收益？

### 3. 个人贡献
- 基于 Git 数据，个人在项目中的角色是什么？
- 主要负责了哪些模块？
- 有哪些关键的代码贡献？

### 4. 简历亮点
- 提取 3-5 个最值得写进简历的亮点
- 每个亮点尽量量化
- 用专业的技术语言描述

## 输出格式

请以 JSON 格式输出分析结果：

```json
{
  "role": "在项目中的角色",
  "keyModules": ["负责的核心模块"],
  "technicalHighlights": ["技术亮点"],
  "businessImpact": "业务价值描述",
  "resumeHighlights": ["简历亮点（可直接使用）"],
  "suggestedTitle": "建议的项目标题"
}
```
